\# 👁️ CNN \& Image Classification Repository



This folder is dedicated to \*\*Computer Vision\*\* projects using \*\*Convolutional Neural Networks (CNN)\*\*. It demonstrates the ability to process visual data and classify images with high precision across various domains.



\### 🧠 Deep Learning Expertise:

\* \*\*Architecture:\*\* Designing custom CNN layers (Convolution, Pooling, Flattening, and Dense Layers).

\* \*\*Optimization:\*\* Using Dropout layers and Batch Normalization to prevent overfitting.

\* \*\*Image Processing:\*\* Data augmentation, scaling, and normalization using OpenCV and TensorFlow.

\* \*\*Classification:\*\* Expertise in both Binary (Happy/Sad) and Multi-class (PlantVillage 38 Classes) classification.



\### 🛠 Frameworks:

\* \*\*Engine:\*\* TensorFlow / Keras

\* \*\*UI:\*\* Streamlit Web Interface

\* \*\*Vision:\*\* OpenCV
