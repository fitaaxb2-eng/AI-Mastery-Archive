# 🍅 Tomato Leaf Disease Detector

A simple Deep Learning project that classifies tomato leaves into two categories: **Healthy** or **Early Blight**. This model was built using Transfer Learning with **MobileNetV2**.

## 🚀 Features
- **Fast Prediction:** Uses a lightweight model optimized for speed.
- **Streamlit UI:** Simple and user-friendly web interface.
- **Image Preprocessing:** Automatically resizes and prepares images for classification.

## 🛠️ Tech Stack
- **Python**
- **TensorFlow/Keras**
- **Streamlit**
- **Pillow & Numpy**
